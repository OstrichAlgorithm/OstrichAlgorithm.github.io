企鹅视频去广告

  • javascript
  • 去广告
funny

首先3Q企鹅，尽管他可能不大乐意
接着那篇企鹅视频的获取接口

又搞出一种方法可以去广告播放

将下面的蓝色按钮拖拽到叫做收藏夹或者是书签栏的地方

腾讯视频去广告

在企鹅视频里，进入你想观看的视频页面，比如《神探狄仁杰 第七集》，然后点击刚才收藏的书签即可去广告观看

因为知道它是用过jsonp来获取视频真实地址，所以可以通过正则表达式来全文查找到vid，再用官方的视频播放器播放书签代码如下，

#或者直接复制下面这段代码到地址收藏栏

javascript:var html = document.body.innerHTML;var reg=/vid:”(\w+)”/g;var r = reg.exec(html);location.href=’http://v.qq.com/iframe/player.html?vid='+r[1];”
