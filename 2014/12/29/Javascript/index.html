javascript

完整的javascript包括三个部分：

  • 核心（ECMAScript）
  • 文档对象模型（DOM）
  • 浏览器对象模型（BOM）

ECMAScript

根据ECMA-262定义的ECMAScript本身没有参照web浏览器标准。
ECMAScript定义的内容为：

  • 语法
  • 类型
  • 语句
  • 关键字
  • 保留字
  • 操作符
  • 对象

DOM

文档对象模型（DOM,Document Object Model）是针对XML拓展于HTML的API。Dom把整个文档映射一个多节点的结构。

  • 核心（DOM Core）
  • 标签（DOM HTML）
  • 视图（DOM Views）：不同文档的视图接口
  • 事件（DOM Events）：事件和事件处理接口
  • 样式（DOM Style）：css
  • 遍历和范围（DOM Traversal and Range）：遍历和操作文档树的接口
  • 保存和加载（DOM Save and Load）:统一加载保存
  • 验证（DOM Validation）：验证文档

BOM

浏览器对象模型（BOM,Browser Object Model）,给浏览器制作的接口，比如cookie,location,screen,window等
