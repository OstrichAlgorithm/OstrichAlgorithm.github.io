ECMAScript的基本数据类型

javascript

数据类型

EMCAScript的数据类型：

  • Undefined
  • Null
  • Boolean
  • String
  • Number
  • Object

Undefined类型

此类型只有唯一值:undefined。

Null类型

Null类型只有一个值：null。空对象指针。

Boolean类型

显而易见的true和false。

任何类型可以和从Boolean()转化为boolean类型。

Number类型

整数

浮点数

数值范围

Number.MIN_VALUE,5e-324

Number.MAX-VALUE,1.7976931348623157e+308

超过范围则转化为特殊值Infinity表示，用户isFinite()检测

NaN
